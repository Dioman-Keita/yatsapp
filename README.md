# yatsapp

## Getting Started

Install dependencies and run the development server:

```bash
bun dev
```

Open the local URL printed by Vite, usually http://localhost:5173.

## Scripts

- `bun dev` starts the Vite/TanStack Start dev server.
- `bun run build` creates the production build.
- `bun run start` previews the production build.
- `bun run lint` runs ESLint.

This app uses TanStack Start, TanStack Router, React, Vite, Tailwind CSS, and Convex.
